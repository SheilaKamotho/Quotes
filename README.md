## Description
This is an application that allows the user to generate goals and read quotes written by others. The quotes can also be deleted, upvoted and downvoted for.

## Contributers
Sheila Kamotho (24/08/2020)

## How it works
Goal is opened on any web-browser and the user can view, submit and like/dislike quotes.

## Installation 
The user is required to have any browser.
